# Valentine Images Directory

Place your Valentine's Day images here:

- `rose-bloom.gif` - Animated rose for greeting stage
- `heart-glow.jpg` - Glowing heart for message stage  
- `love-fireworks.gif` - Celebration fireworks for closing stage

Recommended sizes:
- Greeting image: 200x200px
- Message image: 160x160px
- Celebration image: 224x224px

If images are not found, the experience will use emoji fallbacks.
